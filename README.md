# The Timber Lights (TTL)

A modern interactive mobile web application for **The Timber Lights**. Showcases premium lighting products with an interactive UI, real product images, and a dynamic light toggle feature.

## Features
- **Responsive Design**: Optimized for mobile and desktop viewing.
- **Interactive Lighting**: Toggle product lights ON/OFF to see realistic glow effects.
- **Brand Story**: Learn about the founder and the vision behind the brand.
- **Store Location**: Integrated Google Maps for easy navigation to the physical store.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Icons**: Font Awesome 6.4
- **Fonts**: Outfit (Google Fonts)

## Setup & Running Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (optional, for dev server)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tanishjain93133-hub/TTL
   ```
2. Navigate to the project directory:
   ```bash
   cd TTL
   ```

### Running the App
- **Option 1: Static Files**
  Open `index.html` directly in any web browser.
  
- **Option 2: Development Server**
  If you have Node.js installed:
  ```bash
  npm install
  npm start
  ```

## Environment Variables
This project currently does not require any external environment variables.

Deployed on **Vercel** for production.
